# 图的拓扑排序

## 1.拓扑排序

算法：**Kahn 算法**： 通过记录每个节点的入度，逐步删除入度为 0 的节点，依次输出排序结果。如果在排序结束后，仍有节点未被删除（即剩下有入度的节点），说明图中存在环。
